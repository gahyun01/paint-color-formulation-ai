# Paint Color Formulation AI

> 조색제 개별 색정보가 없는 상황에서, 목표 LAB 색상값으로부터 조색제 배합 비율을 예측하는 AI 기반 조색 모델 Case Study입니다.


## Overview

본 프로젝트는 (주)컴파이드에서 진행한 페인트 조색 AI 모델 개발 프로젝트입니다.

페인트 제조사와의 협업 과정에서 축적된 조색 데이터를 바탕으로, 목표 색상 LAB 값을 입력하면 *개 조색제의 배합 비율을 예측하는 모델을 설계했습니다.

회사 보안 정책상 소스 코드, 원본 데이터셋, 학습된 모델 가중치, 협력사명 및 내부 구현 세부사항은 공개하지 않습니다.  
본 README는 공개 가능한 범위 내에서 문제 정의, 접근 방식, 모델 구조, 주요 기여를 정리한 기술 요약 문서입니다.


## Background

일반적인 조색 과정은 목표 색상에 가까운 배합을 찾기 위해 반복적인 실험과 보정이 필요합니다.

이 프로젝트의 목표는 이러한 시행착오를 줄이기 위해, 목표 LAB 색상값으로부터 조색제 배합 비율을 직접 예측하는 모델을 만드는 것이었습니다.

사용 가능한 데이터는 다음과 같았습니다.

- 조색 후 측정된 최종 LAB 값
- 해당 LAB 값을 만들기 위해 사용된 *개 조색제의 배합 비율
- 4개의 베이스별 조색 데이터
- 배합 비율의 합이 100인 샘플

가장 큰 제약은 *개 조색제 각각에 대한 개별 색정보가 없었다는 점입니다.  
즉, 데이터에는 “어떤 조색제를 얼마만큼 섞었을 때 어떤 LAB 값이 나왔는지”만 존재했고, 각 조색제가 단독으로 어떤 색상 특성을 가지는지는 제공되지 않았습니다.


## Problem Definition

이 프로젝트는 단순히 LAB 값을 입력받아 배합 비율을 예측하는 일반적인 지도학습 문제로 접근하기 어려웠습니다.

조색제 개별 색정보가 없었기 때문에, 먼저 각 조색제의 잠재적인 색상 표현을 추정하고, 이를 기반으로 조색 모델을 구성해야 했습니다.

따라서 문제를 다음과 같이 재정의했습니다.

```text
목표 LAB 값 → 조색제 배합 비율 예측 → 예측 LAB 재구성 → 목표 LAB와 비교
```

즉, 모델이 단순히 비율을 맞히는 것이 아니라, 예측한 비율로 만들어지는 색상이 목표 LAB에 가까워지도록 학습 구조를 설계했습니다.


## Data Preparation

데이터 전처리는 조색 모델의 안정성을 높이기 위해 다음과 같이 진행했습니다.

- LAB 값이 없는 샘플 제거
- 조색제 비율 결측치는 0으로 대체
- 조색제 비율 합이 100인 샘플만 사용
- 동일한 배합 비율 그룹 내에서 LAB 편차가 큰 이상치 제거
- 베이스별로 데이터 분포와 조색 특성이 다르기 때문에 베이스 단위로 데이터 분리
- 색상 영역별 데이터 분포 확인

데이터를 확인한 결과, 대체로 진한 파란색, 보라색, 짙은 녹색 계열의 데이터가 부족했습니다.  
이 색상 영역들은 최종 모델에서도 상대적으로 높은 오차가 발생한 구간이었습니다.


## Colorant Representation Estimation

*개 조색제에 대한 개별 LAB 정보가 없었기 때문에, 먼저 데이터로부터 조색제별 초기 색상 표현을 추정했습니다.

각 조색제의 비율이 높은 샘플을 선별하고, 해당 샘플들의 LAB 중앙값을 사용해 조색제의 초기 LAB 값을 추정했습니다.  
샘플 수가 부족한 경우에는 해당 조색제 비율이 높은 상위 샘플을 사용했습니다.

이후 Kubelka-Munk 이론에서 착안한 색 혼합 방식을 활용해, 추정된 조색제 LAB 값이 실제 조색 결과를 잘 설명하도록 반복적으로 보정했습니다.

이 단계에서 얻은 조색제 LAB 값은 정답값으로 사용한 것이 아니라, 이후 모델 학습을 위한 초기값으로 활용했습니다.


## Forward Model

다음 단계에서는 조색제 배합 비율을 입력받아 LAB 값을 예측하는 forward model을 학습했습니다.

```text
조색제 배합 비율 → LAB 값
```

베이스별로 데이터 분포와 조색 특성이 달랐기 때문에, 모델은 베이스별로 따로 구성했습니다.

forward model은 단순히 배합 비율과 LAB 값 사이의 통계적 관계만 학습하는 것이 아니라, Kubelka-Munk 이론에서 착안한 색 혼합 구조를 함께 반영하는 방향으로 설계했습니다.  
이를 통해 조색제 비율 변화가 최종 LAB 값에 미치는 비선형적인 영향을 더 잘 반영하고자 했습니다.

모델 구성 과정에서는 다음과 같은 방식들을 비교했습니다.

- 기본 신경망 모델
- 여러 seed 기반 Ensemble 모델
- BatchNorm과 Dropout을 적용한 깊은 신경망 모델
- L, a, b에 서로 다른 가중치를 적용한 loss 모델
- ratio, log ratio, ratio², 상위 조색제 비율 등을 활용한 feature engineering 모델
- Kubelka-Munk 이론에서 착안한 색 혼합 구조 반영

모델 평가는 주로 ΔE76을 기준으로 진행했습니다.


## Inverse Model

forward model을 학습한 뒤에는 이를 고정된 시뮬레이터처럼 사용하여 inverse model을 학습했습니다.

```text
목표 LAB 값 → 조색제 배합 비율 → forward model → 예측 LAB 값
```

inverse model은 목표 LAB 값을 입력받아 *개 조색제의 배합 비율을 출력합니다.  
출력단에는 softmax를 사용하여 전체 조색제 비율의 합이 1이 되도록 구성했습니다.

또한 LAB 입력값은 그대로 사용하지 않고, 다음과 같은 9차원 feature로 확장했습니다.

```text
L, a, b, L*a, L*b, a*b, L², a², b²
```

이 구조를 통해 모델은 목표 LAB 값과 조색제 비율 사이의 비선형 관계를 더 잘 학습할 수 있도록 했습니다.


## Optimization

inverse model이 예측한 비율만으로 목표 오차 기준을 만족하지 못하는 경우, 추가 최적화를 수행했습니다.

전체 예측 흐름은 다음과 같습니다.

```text
목표 LAB 입력
→ inverse model로 초기 배합 비율 예측
→ forward model로 예측 LAB 계산
→ ΔE76 계산
→ 목표 오차 초과 시 gradient 기반 ratio 최적화
→ 필요 시 global search로 보정
→ 최종 배합 비율과 예측 LAB 반환
```

이 방식은 기존 회사 모델보다 훨씬 빠르게 결과를 도출할 수 있었습니다.


## Evaluation

평가 지표는 ΔE76을 기준으로 사용했습니다.  
협력사 측 목표 기준은 ΔE76 0.5 이하였습니다.

모델 확인 결과, 데이터가 충분히 존재하는 대부분의 색상 영역에서는 ΔE76 0.5 이하를 달성했습니다.  
확인한 샘플 중 가장 낮은 오차는 약 0.004 수준이었습니다.

반면 데이터가 부족했던 진한 파란색, 보라색, 짙은 녹색 영역에서는 오차가 크게 발생했습니다.  
해당 구간에서는 ΔE76이 약 60 수준까지 나타났지만, 기존 모델의 약 80 수준 오차와 비교하면 개선된 결과였습니다.

또한 기존 모델은 높은 오차를 최적화 알고리즘 기반으로 강제 수렴하는 방식이었기 때문에 약 1시간 30분에서 2시간 정도가 소요되었습니다.  
반면 본 프로젝트에서 구성한 모델은 일반적인 시뮬레이션 기준 1분 이내에 결과를 반환할 수 있었습니다.


## Key Contributions

- 조색제 개별 색정보가 없는 데이터 상황에서 문제를 재정의
- 조색 결과 LAB 값과 배합 비율만으로 *개 조색제의 초기 색상 표현 추정
- Kubelka-Munk 이론에서 착안한 색 혼합 방식으로 초기 조색제 표현 보정
- Kubelka-Munk 기반 색 혼합 특성을 forward model 구조에 반영
- 베이스별 데이터 분포 차이를 고려한 모델 파이프라인 구성
- 조색제 비율에서 LAB를 예측하는 forward model 학습
- 목표 LAB에서 조색제 비율을 예측하는 inverse model 학습
- forward model을 고정된 시뮬레이터로 활용해 LAB → ratio → LAB 구조 구성
- ΔE76 기준으로 모델 성능 검증
- 기존 모델 대비 추론 시간을 크게 단축
- 데이터 부족 색상 영역을 분석하고 모델 한계 구간을 식별
